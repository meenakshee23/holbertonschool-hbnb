# holbertonschool-hbnb
This is my repository as a full-stack engineer

## Teamwork: HBNB Project
In Part 1, design diagrams for the presentation, business, and persistence layers.


In Part 2,  create the project structure, including all the necessary directories and files. add a models folder, which serves as the core directory where data structures are stored. In this folder, define the main entities: User, Place, Amenity, and Review.



In Part 3, the extend the backend of the application by introducing user authentication, authorization, and database integration using SQLAlchemy and SQLite for development. Later, you'll configure MySQL for production environments. In this part, secure the backend, introduce persistent storage, and prepare the application for a scalable, real-world deployment.




Part 4 - Simple Web Client
In this phase, the focus is on the front-end development of the
application using HTML5, CSS3, and JavaScript ES6. The task is to design and implement an interactive user interface that connects with the back-end services we have developed in previous parts of the project.
